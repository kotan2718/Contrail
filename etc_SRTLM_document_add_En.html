SRTLM Additional Features Manual

This document describes the new features added to the 'SRTLM lens meter simulation tool.'

For the SRTLM instruction manual, please refer to SRTLM instruction manual.

Please download the latest version from SRTLM Lens Meter Simulation Tool.
Clicking "Download will always get you the latest version.

Changes

  • [Lens Data] section: Added checkbox [1.523] (inside red frame)

SRTLM version 2.27.0 startup screen

Figure 1 shows the startup screen of SRTLM version 2.27.0. The only visual change is the addition of this checkbox.


Fig. 1 Screen Layout and Additional Features

When checked, the front surface radius (r1) will be calculated with a refractive index of 1.523, regardless of the refractive index of the lens being tested.
When unchecked, the front surface radius (r1) will be calculated with the refractive index of the lens being tested, as before.

When unchecked

When checked

The ON/OFF status of the checkbox is also written to the data file when saving lens data. However, existing data files (without the new field) remain backward-compatible — they are treated as "unchecked" by default.
