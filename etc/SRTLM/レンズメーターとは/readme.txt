
SRTLM インストールマニュアル

1. SRTLM.zipを解凍すると[SRTLM]フォルダーが現れます。

2. [SRTLM]フォルダー内の構成は以下の通りです。
   [SRTLM] - [dat] - xxxx.srt(サンプルデータが格納されています)
             [exe] - SRTLM.exe(シミュレーションツール本体)
             [lib_img] - [etc]
                         [SRTLM_img] - xxxx.jpg(説明用の図が格納されています)
             SRTLM_about_LensMeter.html(レンズメーターについての解説)
             SRTLM_document.html(このツールのドキュメント)
             readme.txt(このファイルです)

3. SRTLMの起動
   [SRTLM] - [exe] - SRTLM.exe を実行して下さい。

4. 注意
   [SRTM]フォルダーはどこに移動してもかまいません。
   ただし、SRTLMはデータの保存、呼び出しで[dat]フォルダーを使用します。
   [SRTM]フォルダーを移動しても[dat]フォルダーは削除しないで下さい。

5. このシミュレーションツールでは、レジストリへの操作はありません。
