Example of a Limit Cycle 3 Lemniscate

By default, the parameters a, b are set to 1, and c is set to -1.

This autonomous system has three equilibrium points: the origin (saddle point) and two stable vortices (+/- 1 / sqrt{2}, 0).

There are three limit cycles on the curve called the Lemniscate (x^2 + y^2)^2 - (x^2 - y^2) = 0.
The origin serves as an equilibrium point, and the sections where $x < 0$ and $x > 0$ become solution trajectories of the equation.
According to Mr.  Morimoto, these are not limit circles.
This seems to be because two limit cycles are connected.  The limit cycles themselves are stable.

What happens when $a = -1, b = -1, c = 1$?
Please confirm.  The limit cycle becomes unstable.

What about when $a = -1, b = 1, c = -1$?
The three equilibrium points change to the origin (vortex) and two saddle points (+/- 1 / sqrt{2}, 0).
It ceases to be a Lemniscate, and a trajectory resembling the human eye appears around the origin.
For $a = 1, b = -1, c = 1$, the direction of the trajectory is reversed.
In both cases, although I have plotted and observed repeatedly, I believe there might be no limit cycle.

<p class="pb10">
    The Hamiltonian equations have the following characteristics:
</p>
<p class="pb10">
    \[ \frac{dx}{dt} = F_y(x, y)\]
    \[ \frac{dy}{dt} = -F_x(x, y)\]
    \[ The analytical solution becomes the implicit function as follows: \]
    \[ F(x, y) = C\]
</p>
<p class="pb10">
    Since the analytical solution becomes an implicit function,
    directly integrating to find a general solution can be unexpectedly tricky.
    One approach is to divide the drawing area into a grid and, without modifying the function,
    approximate the graph by plotting while checking if the four sides of each grid intersect with the function.
</p>

解析解を求めると陰関数となるので、一般解を描画するのは以外とやっかいである。
媒介変数を使って表示するという方法もあるが、求められた陰関数に手を加えず描画するという横着な考え方をすると、
描画エリアを格子状に分割して、各格子の4辺と陰関数が交わるか調べながら近似的にグラフをプロットしていくというのが、簡単な方法ではないかと思う。

微分方程式の解軌道は、解析解の曲線族をプロットすることになるので、
Hamilton型の場合は、ダイレクトに陰関数のグラフを描画することになる。

陰関数のグラフ "Cassiniの燈形" と一致することが確認できる。
陰関数のグラフの定数が0.25のときはレムニスケートになる。

Since the analytical solution becomes an implicit function, it is relatively difficult to plot the general solution.
Since the analytical solution becomes an implicit function, it is relatively difficult to draw the general solution.
While one method involves using parameterization for display,
lazily opting to plot without modifying the derived implicit functions leads to a cumbersome process.
A simpler approach might involve dividing the drawing area into a grid and approximating the graph
by plotting while checking if the implicit function intersects with the four sides of each grid.

The trajectory of solutions to the differential equation entails plotting the curve family of analytical solutions.
Thus, in the case of Hamiltonian equations, directly plotting the graph of implicit functions is necessary.

Confirmation can be made by observing that the graph of the implicit function "Cassini's oval" matches.
When the constant of the implicit function graph is 0.25, it becomes a lemniscate.
This is markedly different from what was previously introduced in [4]-6,
where the general solution of the lemniscate was shown to be a limit trajectory.
